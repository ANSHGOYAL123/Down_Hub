## 📦 Installation Guide

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/Down_Hub.git
cd Down_Hub
```

---

### 2️⃣ Create Virtual Environment

```bash
python -m venv venv
```

Activate:

```bash
venv\Scripts\activate
```

---

### 3️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

---

### 4️⃣ Install FFmpeg (Important)

Download from: https://ffmpeg.org/download.html

Verify installation:

```bash
ffmpeg -version
```

---

### 5️⃣ Run the App

```bash
streamlit run app.py
```

---

## ⚠️ Known Limitations

* Instagram may block some formats → fallback mode used
* Some videos may not provide thumbnails
* YouTube may show JS runtime warning (optional fix: install Node.js)
